# Portfolio Construction & Monte Carlo Optimization

A quantitative finance project implementing portfolio construction, asset allocation, and Monte Carlo-based portfolio optimization using Modern Portfolio Theory (MPT).

The framework maximizes the Sharpe ratio and visualizes the efficient frontier through interactive simulations.

# Features

- Financial data preprocessing and return computation
- Random portfolio weight generation using Dirichlet distribution
- Asset allocation and portfolio value tracking
- Expected annual return and volatility estimation
- Sharpe ratio computation and optimization
- Monte Carlo simulation (10,000+ portfolios)
- Efficient Frontier visualization
- Interactive Plotly-based analytics

# Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Plotly
